# hls-outreach-sms
Outreach Solution Blueprint for Healthcare Setting

###Pre-requisites

- Twilio CLI
  - serverless toolkit
- git CLI

###Installation Steps

```shell
git clone https://github.com/twilio/hls-outreach-sms.git
```

```shell
cd hls-outreach-sms
```

Edit your `.env` variables values.

```shell
twilio serverless:deploy
```
